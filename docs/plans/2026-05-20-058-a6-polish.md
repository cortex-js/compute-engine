## Task 3: `\operatorname{repeat}` arity

Today's behavior: `Repeat`'s library signature is `(value: any) -> list` (infinite sequence). Desmos's `\operatorname{repeat}(value, count)` has 2 args and is a finite repeat. CE's existing lowercase alias `\operatorname{repeat} → Repeat` then errors on the second arg with `unexpected-argument`.

Audit context: 1 corpus row.

**Design decision (made here, not deferred):** Widen `Repeat`'s signature to `(value: any, count?: integer) -> list`. When `count` is provided, return a finite list; when absent, keep the existing infinite-sequence semantics. This avoids introducing a new operator name and matches Desmos's behavior.

```typescript
  Repeat: {
    description:
      'Produce a sequence by repeating a single value. With 1 argument, returns an infinite sequence; with 2 arguments (value, count), returns a finite list of `count` copies.',
    complexity: 8200,
    signature: '(value: any, count: integer?) -> list',
    collection: {
      isLazy: (expr) => expr.ops?.length === 1,
      count: (expr) => {
        if (expr.ops?.length === 2) {
          const n = expr.op2.evaluate();
          if (n.isInteger && n.isFinite) return Number(n.numericValue);
        }
        return Infinity;
      },
      isEmpty: (expr) => {
        if (expr.ops?.length === 2) {
          const n = expr.op2.evaluate();
          if (n.isInteger) return Number(n.numericValue) <= 0;
        }
        return false;
      },
      isFinite: (expr) => expr.ops?.length === 2,
      contains: (expr, target) => {
        if (!isFunction(expr)) return false;
        return expr.op1.isSame(target);
      },
      iterator: (expr) => {
        if (!isFunction(expr))
          return { next: () => ({ value: undefined, done: true }) };
        if (expr.ops?.length === 2) {
          const n = expr.op2.evaluate();
          const count = n.isInteger && n.isFinite
            ? Math.max(0, Number(n.numericValue))
            : 0;
          let i = 0;
          return {
            next: () => i++ < count
              ? { value: expr.op1, done: false }
              : { value: undefined, done: true },
          };
        }
        return { next: () => ({ value: expr.op1, done: false }) };
      },
      at: (expr, index) => {
        if (!isFunction(expr)) return undefined;
        if (expr.ops?.length === 2) {
          const n = expr.op2.evaluate();
          if (n.isInteger && n.isFinite) {
            const count = Number(n.numericValue);
            if (index < 1 || index > count) return undefined;
          }
        }
        return expr.op1;
      },
    },
  },
```

Note: the `at` index here is 1-based (matching CE's collection convention). Verify by looking at neighboring operators in the same file before finalizing.

  normalizeIdentifier(latex: string): string {
    if (!latex) return '';
    // Parse in a temporary scope so any auto-declared symbol doesn't
    // leak into the engine scope chain.
    this.pushScope();
    try {
      const expr = this.parse(latex);
      return expr.symbol ?? '';
    } finally {
      this.popScope();
    }
  }
